\section{Resolution} 

  The scope refers to where a name is visible. 

  \begin{definition}[Scope]
    The \textbf{scope} of a variable is the region of code where the name is visible. 
  \end{definition}

  The resolution is basically a syntactical analysis step where we make sure which value does a given variable refer to? We call this \textit{resolving a variable}. 

  Later on, we will see this implemented through an \textbf{environment}, which is a runtime data structure that stores the name-to-value bindings.
